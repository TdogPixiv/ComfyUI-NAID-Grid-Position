# NAID Grid Position (ComfyUI)

A ComfyUI node that outputs X/Y floats (0.1..0.9) using a clickable 5x5 grid UI, with an optional Random toggle that randomizes the grid selection each queue.

## Install (manual)
1. Copy this repo folder into: `ComfyUI/custom_nodes/naid_grid_position/`
2. Restart ComfyUI (hard refresh browser if needed)

## Install (ComfyUI-Manager)
Add this repository to ComfyUI-Manager (see below).
